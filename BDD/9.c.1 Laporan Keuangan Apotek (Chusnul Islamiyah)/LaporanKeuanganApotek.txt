Scenario 1: user melihat laporan keuangan apotek Tahunan
Given	: user berada dalam menu laporan keuangan apotek
When	: memilih jenis laporan Tahunan
Then	: Interface Laporan keuangan membutuhkan Tahun yang tersedia di class laporan Keuangan Apotek Entitas
And	: class Laporan Keuangan Apotek Entitas melakukan return (Tahun)
When	: memilih Tahun
Than	: Interface Laporan Keuangan membutuhkan list <keuangan>
And	: class queryLaporanKeuanganApotek memanggil setData yang berisi (bulan dan total pendapatan ) pada laporanKeuanganApotekEntitas
And	: queryLaporanKeuanganApotek addlist
And	: queryLaporanKeuanganApotek mereturn lkeuangan
And	: interface laporankeuanganApotek setModel yang berisi Tabellaporan keuanganApotek
When	: menekan tombol cetak
And	: mengconvert to PDF
